🚀 Cursor + OpenRouter Setup for LLM KnightTrader

Free setup with Owl Alpha model — get running in ~5 minutes

1
Install Cursor

Download and install Cursor (free):

Go to: cursor.com/download
Windows: run the .exe installer
Mac: drag .dmg to Applications

Open Cursor and sign in with your free Cursor account.

2
Get OpenRouter API Key (Free)

1. Go to openrouter.ai/keys

2. Sign up / log in

3. Click "Create Key" — copy the key (starts with sk-or-...)

Free tier gives you up to 1,000 requests/day with Owl Alpha. No credit card needed.
3
Configure Cursor for OpenRouter (Custom Base URL)

In Cursor, open Settings:

Windows: Ctrl + Shift + J
Mac: Cmd + Shift + J

Navigate to Models section, then click the API Keys dropdown arrow:

Set these values:

Base URL: https://openrouter.ai/api/v1
API Key:  sk-or-your-key-here
Model:   openrouter/owl-alpha
If Cursor doesn't have a custom provider option in settings, you can set it via environment variables or MCP config. See note below.

Alternative method — set environment variables before launching Cursor:

# Windows (PowerShell):
$env:OPENAI_API_KEY = "sk-or-your-key-here"
$env:OPENAI_BASE_URL = "https://openrouter.ai/api/v1"

# Mac/Linux:
export OPENAI_API_KEY="sk-or-your-key-here"
export OPENAI_BASE_URL="https://openrouter.ai/api/v1"
4
Enable "Run Everything" Mode (Auto-Run)

This lets Cursor work autonomously without asking you to click "Run" every time:

1. Open Cursor Settings (Ctrl/Cmd + Shift + J)
2. Go to Agents → Approvals & Execution
3. Find "Auto-Run" mode
4. Select "Run Everything"
⚠️ "Run Everything" lets the agent execute any terminal command without approval. Only use in trusted environments. Cursor creates checkpoints before changes so you can roll back if needed.

Now Cursor will run commands, install packages, and execute scripts automatically — no manual approval needed.

5
Clone and Run LLM KnightTrader

Open a terminal in Cursor and run:

git clone https://github.com/1bananaonthewall-ux/llm-knighttrader.git
cd llm-knighttrader
python -m venv .venv

# Windows:
.venv\Scripts\activate

# Mac/Linux:
source .venv/bin/activate

pip install -r requirements.txt

Then paste this prompt into Cursor chat:

Read ReadMe.tx in full, then set up this project.
Ask me for my BloFin API keys and OpenRouter key in chat.
Use OpenRouter with base URL https://openrouter.ai/api/v1 and model openrouter/owl-alpha.
After setup, verify end-to-end, then STOP the stack.
Enable auto-run so you don't need my approval for commands.

Cursor will ask for your keys, then handle everything automatically — install, configure, verify, and stop.

6
Daily Use

After setup, you'll have desktop shortcuts:

Start LLM KnightTrader.lnk  → kills old instances, starts fresh
Stop LLM KnightTrader.lnk   → stops everything

Or from command line:

python scripts/stack_launcher.py start
python scripts/stack_launcher.py stop
